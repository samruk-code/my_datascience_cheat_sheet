# Data Science Cheat Sheet Collection

A personal reference library of Jupyter notebooks covering the core tools and workflows used in data science and machine learning. Built as a quick-lookup resource — each notebook focuses on practical, frequently used code patterns rather than theory.

---

## Notebooks

| Notebook | Description |
|---|---|
| [intro_python.ipynb](intro_python.ipynb) | Python fundamentals — syntax, data types, control flow, and functions |
| [python_tool_box.ipynb](python_tool_box.ipynb) | Python data science toolbox — iterators, list comprehensions, and functional programming |
| [importing_data.ipynb](importing_data.ipynb) | Loading data from flat files, databases, and web sources |
| [pandas_sample_codes.ipynb](pandas_sample_codes.ipynb) | Data manipulation with pandas — filtering, grouping, merging, and reshaping |
| [Ml Python Cookbook.ipynb](Ml%20Python%20Cookbook.ipynb) | NumPy vectors, matrices, and array operations for ML workflows |
| [data_visualization_with_python.ipynb](data_visualization_with_python.ipynb) | Visualization with Matplotlib — plots, subplots, and customization |
| [SQL Fundamentals.ipynb](SQL%20Fundamentals.ipynb) | SQL essentials — SELECT, JOIN, GROUP BY, subqueries, and window functions |
| [supervised_ml_scikit_learn.ipynb](supervised_ml_scikit_learn.ipynb) | Supervised learning with scikit-learn — classification, regression, and model evaluation |
| [ML Engineer.ipynb](ML%20Engineer.ipynb) | MLOps concepts — deploying and maintaining ML models in production |
| [data_engineers_short_notes.ipynb](data_engineers_short_notes.ipynb) | Short notes on data engineering concepts and pipelines |
| [git.ipynb](git.ipynb) | Git workflow reference — branching, merging, rebasing, and common commands |

---

## Usage

Clone the repo and open any notebook with Jupyter:

```bash
git clone https://github.com/samruk-code/my_datascience_cheat_sheet.git
cd my_datascience_cheat_sheet
jupyter notebook
```
